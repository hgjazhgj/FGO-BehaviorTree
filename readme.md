# Fate/Grand Order行为树

## 解决的问题

解决FGO中根据某回合的发牌等因素决定后续战斗行为的问题,先后包括:

1. 「差分选项」出现某情况就进某条世界线  
2. 「好感度选项」通过历史出现过的情形在判定节点决定世界线走向  
3. 后效性决策  

## 计划实现的功能

+ FGO特化的行为树结构  
+ 可视化的Web行为树编辑前端  
+ 将行为树编译为FGO-py可用的Battle类  

前端应该能够从哪里抄一个过来随便改改就行,主要是xml向python的转换
